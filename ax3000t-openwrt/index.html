小米AX3000T 刷 OpenWrt

刷机

直接按照官方文档操作即可 https://openwrt.org/inbox/toh/xiaomi/ax3000t

传文件会遇到 /usr/libexec/sftp-server: not found when using scp

加 -O 选项，参考 https://forum.openwrt.org/t/ash-usr-libexec-sftp-server-not-found-when-using-scp/125772 解决

配置网络

无线桥接

设置 lan interface 的 IP ，不要和主WIFI在同一子网中 在 Network -> Wireless 中扫描添加，注意 firewall zone 的选项不能删除 添加 wwlan6 interface 设置 IPv6，lan 能自动获取到 IPv6

无线中继是在同一网段，似乎要装 relay 插件，有点麻烦

有线连接无线交换机

设置好 lan 静态 IP，关闭 dhcp，设置 lan6 dncpv6 client

参考 https://forum.turris.cz/t/using-static-local-ipv4-how-to-get-a-public-ipv6-address-too/20366/5

二级路由

SLAAC

配置静态 lan ip

wan6 获取到 IPv6 后，lan 关闭 dhcpv6，开启 slaac

等待一段时间

参考 https://chenhe.me/post/openwrt-config-ipv6-public-access

Relay

把 lan 和 wan 的 DHCP ipv6 相关的设置都设为 relay

wan 勾选 Designated master

参考 https://www.lategege.com/?p=676

信号弱

设置 Operating frequency mode 为 AX

调整 Country Code、Maximum transmit power、Operating frequency

开启 WIFI 漫游（Roaming）

定时任务

直接用 crontab -e，在配置页中也能看到
